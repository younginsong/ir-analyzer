Aether

FTIR peak deconvolution, elevated.

Named after the primordial god of light — Aether brings clarity to complex infrared spectra. Baseline correction, automatic peak detection, and multi-peak fitting for in situ IR spectra, all in one elegant desktop app.

🍎 macOS · 🪟 Windows · 🐧 Linux

Features

Built for
in situ IR analysis.

🎯
Automatic 4-Peak Detection
Second-derivative analysis automatically locates all four sub-peaks in broad, overlapping OH stretching bands — no manual guessing required.
2nd derivative
📊
Population Analysis + Stark Tuning
View peak positions, areas, and area% across the full potential range simultaneously. Calculate population shifts and Stark tuning slopes (cm⁻¹/V) directly from deconvolution — no post-processing needed.
stark tuning · population
🔬
OH and CO Analysis
Supports both OH stretching region and CO stretching region (COL, COB). Switch between analysis modes to cover the full in situ IR workflow in a single tool.
OH · CO_L · CO_B
📂
Batch Processing
Load an entire folder of potential-dependent spectra at once. Apply shared fitting parameters across all files and export every result to a single Excel workbook in one click.
xlsx export
🖱️
Interactive Peak Editing
Click to add peaks, drag markers to adjust center positions, right-click to fine-tune FWHM. Full control without any scripting.
drag & drop
📐
Gaussian · Lorentzian · Voigt
Choose peak shape per experiment. lmfit-powered optimization with live R² display and full error estimation for every fitted parameter.
lmfit engine
