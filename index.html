Features

Everything you need,
nothing you don't.

🎯
Auto Peak Detection
Second-derivative analysis automatically finds hidden peaks in broad, overlapping OH stretching bands — even when they're not visible to the eye.
2nd derivative
🖱️
Interactive Peak Editing
Click on the plot to add peaks. Drag vertical markers to adjust center position. Right-click to fine-tune FWHM. No command line, no scripting.
drag & drop
📐
Gaussian · Lorentzian · Voigt
Choose your peak shape per experiment. lmfit-powered optimization with full error estimation for each fitted parameter.
lmfit engine
⚡
Stark Tuning Analysis
Load in situ potential-dependent spectra and instantly get linear regression slopes (cm⁻¹/V) and R² for each peak — P1 through P4.
linear regression
📂
Batch Processing
Drag a whole folder of CSV files onto the sidebar. One-click batch fitting with shared initial parameters. Export all results to Excel in seconds.
xlsx export
🎨
Dark-first Interface
Designed for long analysis sessions. Multi-spectrum sidebar, live residual display, and color-coded peaks that are actually readable.
pyqtgraph
